# קובצי שמע — Sound Match

האתר רץ גם בלי הקבצים האלה (הם אופציונליים — חוסר קובץ = שקט, ללא שגיאה).
שים קבצים אמיתיים לפי המבנה הבא.

## 🎵 מוזיקה (`music/`)

**חוק חשוב:** כל קובצי הלופ חייבים להיות באותו אורך מדויק = `MUSIC_LOOP_SECONDS`
(ברירת מחדל 16 שניות, מוגדר ב-`src/config/music.config.ts`). זה מה שמאפשר את
המעבר הטבעי בין מסכים (הלופ החדש ממשיך מאותו מיקום-זמן של הישן).

### מוזיקת תפריט — 4 קבוצות
בכל קבוצה אפשר לשים כמה קבצים; אם יש יותר מאחד, נבחר אקראית בכל מעבר.
```
music/menu/home/01.mp3, 02.mp3, ...          ← מסך הבית
music/menu/settings/01.mp3, ...               ← מסך הגדרות
music/menu/instructions/01.mp3, ...           ← הוראות + בחירת פריטים
music/menu/category/01.mp3, 02.mp3, ...        ← אחרי בחירת קטגוריה
```

### מוזיקת משחק — שכבות (tiers)
מוגדר ב-`GAME_MUSIC_TIERS`. כל שכבה מנגנת `plays` מחזורי-לופ (אקראי מתוך הקבצים שלה)
לפני מעבר לשכבה הבאה; אחרי השכבה האחרונה חוזרים לראשונה.
```
music/game/tier1/01.mp3, 02.mp3   ← התחלה רגועה
music/game/tier2/01.mp3            ← אמצע
music/game/tier3/01.mp3, 02.mp3   ← שיא
```
להוספת קובץ/שכבה — ערוך את `src/config/music.config.ts`.

## 🔊 אפקטים (`sfx/`)
ראה `src/config/sfx.config.ts` לרשימה המלאה. אפשר לספק כמה וריאציות
לכל אפקט (נבחר אקראית). שמות הקבצים תואמים לערכים שם.

## 🗣️ הקראה (`tts/`)
נוצר אוטומטית ע"י `npm run tts` (Google TTS). אל תערוך ידנית.
```
tts/he/<id>.mp3   ← עברית
tts/ar/<id>.mp3   ← ערבית
```
